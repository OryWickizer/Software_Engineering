# Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Software_Engineering/proj2/Ecobites
   ```
2. Run the Client:
   ```bash
   cd client
   npm install
   npm run dev
   ```
3. Run the Server:
   ```bash
   cd server
   npm install
   npm run dev
   ```

   - Frontend: http://localhost:5173
   - Backend: http://localhost:3000
